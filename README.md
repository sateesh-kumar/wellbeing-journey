# Well being journey Project

A PHP + PostgreSQL application for conducting holistic wellbeing journey audit with recommendations to improve your wellbeing.

## Prerequisites

- PHP (v8 or higher)
- PostgreSQL (v14 for database)


The API server will run on `http://localhost/`
